Kasir Access

Kasir Access is a comprehensive sales and product management application designed for minimarkets and supermarkets. It offers features such as transaction management, inventory tracking, sales reports generation, user and product data management, and printing capabilities for receipts and reports. Built with user-friendly interfaces, Kasir Access streamlines the process of managing sales and products, enhancing efficiency for businesses in the retail sector.

Key Features:

Admin Panel:
  • Manage User Data: Admin can manage user data. Including regristration, deletion, update, search user data.
  • Manage Product Data: Admin can manage product data. Including add, update, upload, delete, search of product data.
  • Manage Report: Admin can search, print log data transaction for documentation.
  • Transaction Panel: Admin can add and print struct transaction.

    • Key Features :

    Owner Panel:
  • Manage Report: Owner can search, print log data transaction for documentation.
  • Transaction Panel: Owner can add and print struct transaction.

    • Key Features :

    Cashier Panel:
  • Transaction Panel: Cashier can add and print struct transaction.
