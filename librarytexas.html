Texas Library

Texas Library is a desktop-based application designed specifically to manage library data at SMKS Texar Karawang. The application provides two levels of access, namely admin and workers, each with different functions according to their respective roles.

Key Features:

Admin Panel:
  • Manage Book Data: Admin can manage book data. Including add, update, delete, search, upload book data.
  • Manage User Data: Admin can manage user data. Including regristation, deletion, upload of user data.
  • Manage Borrow Data: Admin can add, update, delete borrow data to log it.

    • Key Features :

    Workers Panel:
  • Manage Borrow Data: Workers can add, update, delete borrow data to log it.
  • Manage Book Return Data: Workers can update borrow data to log it as a return.

    • Review Video

    Outcome :

    Texas Library has been successfully implemented and used for manage library at SMKS Texar Karawang. The application facilitates admin in managing books, users, borrow, and book return. Also workers in managing borrow and book return
