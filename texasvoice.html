TexasVoice

TexasVoice is a web-based application designed specifically for the election of the student council president at SMKS Texar Karawang. The application provides two levels of access, namely admin and voter, each with different functions according to their respective roles.

Key Features:

Admin Panel:
  • Voter Data Management: Admin can manage voter data, including registration and deletion of voters.

  • Candidate Management: Admin can add, edit, or delete data for student council president candidates.

  • Vote Result Monitoring: Admin has access to view real-time election results.

    • Key Features :

    Voter Panel:
  • Voting Option: Voters can cast their votes for their preferred candidates.

  • Candidate Management: Admin can add, edit, or delete data for student council president candidates.

  • User-Friendly Interface: A friendly user interface design makes it easy for voters to participate in the election.

    • Challenges & Sollutions

    Data Security :
  • Challenge: Ensuring the security of voter data and vote results.
  • Solution: Implementing security protocols such as data encryption and protection againts security attacks.

    • Responsive User Interface :
  • Challenge: Providing a responsive user interface for various devices.
  • Solution: Using responsive design and conducting thorough cross-device testing.

    • Review Video

    Outcome :

    TexasVoice has been successfully implemented and used during the student council president election at SMKS Texar Karawang. The application facilitates admin in managing voters and candidates, providing a convenient voting experience for participants.
